# 🇯🇵 Japan Knowledge Library

A comprehensive digital library system dedicated to exploring books about Japan's history, culture, food, and traditions. Built with modern web technologies and featuring beautiful UI/UX design.

## 🌟 Features

### 🔐 Authentication System
- **QR Code Authentication** with bypass option
- **Three.js Loading Screen** with credits
- **Secure session management** with 24-hour expiry
- **Smooth transitions** between authentication states

### 📚 Comprehensive Book Collection
- **20 Specialized Categories** covering Japanese topics
- **Enhanced Google Books API** integration
- **Curated content** for Japanese history, food, and culture
- **Advanced search** with Japanese-specific queries

### 🎨 Beautiful UI/UX
- **Modern React + TypeScript** architecture
- **Tailwind CSS** with Japanese-themed colors
- **Framer Motion** animations and transitions
- **Responsive design** for all screen sizes
- **Toast notifications** for user feedback

### 🎭 Three.js Experience
- **3D particle animations** during loading
- **Rotating torus knot** with Japanese red colors
- **Dynamic lighting** and WebGL rendering
- **Credits screen** for Dexter C. Carrreon

## 🚀 Getting Started

### Prerequisites
- Node.js (v16 or higher)
- npm or yarn

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/yourusername/japan-knowledge-library.git
   cd japan-knowledge-library
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start development server**
   ```bash
   npm run dev
   ```

4. **Open your browser**
   Navigate to `http://localhost:5175`

### Build for Production

```bash
npm run build
```

## 🔧 Configuration

### Google Books API
The application uses the Google Books API with enhanced search queries specifically curated for Japanese content:

- **History**: Ancient samurai, feudal age, Tokugawa era
- **Food**: Sushi techniques, ramen culture, washoku cooking
- **Culture**: Zen Buddhism, festivals, tea ceremonies
- **Modern**: Anime, business, urban Japanese life

### Environment Variables
Create a `.env.local` file for local development:
```env
VITE_GOOGLE_BOOKS_API_KEY=your_api_key_here
```
